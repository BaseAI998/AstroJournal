# AstroJournal 页面设计（复古中世纪羊皮纸）

## 全局设计规范

### Layout

* 移动优先：默认就是一页全屏 Capture；底部仅 2 个 Tab（日记/星盘）。

* 桌面适配：仍保持单栏居中（最大宽 760–860），不引入侧栏导航。

* 对齐：输入区域以左对齐为主，避免卡片化；信息如古老卷轴上的注脚。

### Meta Information

* Title：AstroJournal

* Description：复古中世纪风格的离线优先日记与情绪回顾工具

* Open Graph：

  * og:title = AstroJournal

  * og:description = 在羊皮卷轴上记录你的宇宙航迹

  * og:image = 羊皮纸纹理背景 + 墨迹线条（品牌图）

### Global Styles（Design Tokens）

* 背景：#F4EBD0（羊皮纸米黄）

* 面板：#E6D5B8（深色羊皮纸/木质感）

* 边框/分割线：rgba(0,0,0,0.15)（淡墨迹）

* 主文本：rgba(0,0,0,0.86)（深黑墨水）

* 次文本：rgba(0,0,0,0.54)（褪色墨水）

* 强调色（古金）：#B8860B

* 辅助强调（乌贼墨）：#704214

* 危险色：#8B0000

* 字体层级：

  * H1 22–24 / SemiBold / Serif

  * H2 16–18 / Medium / Serif

  * Body 14–15 / Regular / Serif

  * Caption 12 / Regular / Serif

* 按钮：圆角 8；默认填充面板色；带极细边框模拟手工裁剪；点击时增加内阴影。

* 链接：古金色，hover 下划线。

* 动效：200ms easeOut；页面切换如翻书般淡入；避免电子感过强的发光。

## Page 1：日记 Capture（默认入口）

### Layout

* 全平台：全屏输入为中心；不出现列表、不出现卡片流。

* 顶部仅保留一行极小文案与一个极小“+”图标。

### Page Structure

1. 顶部微栏（装饰/衬托）
2. 全屏输入框（主角）
3. 底部导航（2 Tab）

### Sections & Components

* 顶部微栏

  * 左侧：极小字体星象文案（例：此刻月亮在双子座）

  * 右侧：极小“+”图标（进入历史图），低对比度墨迹风格

  * 可选：再上一行更小的装饰句（如“当下即一切，历史皆墨迹”）

* 全屏输入框

  * 无边框、多行、自动聚焦（可选）；placeholder：此刻有什么想留下的？

  * 输入时不出现格式化工具条，不出现标题/标签/情绪等额外控件

  * 发送按钮：右下角或键盘“发送/完成”，点击后触发墨迹晕染动效并清空输入

  * 保存反馈：不弹 Toast；仅在顶部微栏短暂闪现“已录入”/墨点掠过

* 底部导航（2 Tab）

  * 日记（默认高亮）/ 星盘

  * 图标线性、低对比；选中态出现古金色微光

## Page 2：历史图（History Graph）

### Layout

* 全平台：上半部折线图，节点点击后以弹出卡片形式展示概览。

* 交互以“点选节点→展示卡片→点击详情→进入日记详情页”完成，不引入列表。

### Page Structure

1. 顶部极简栏（返回/关闭）
2. 横向滚动折线图
3. 节点概览卡片（动画弹出）

### Sections & Components

* 折线图

  * X 轴：时间；Y 轴：综合运势分（0..100 或 -100..100）

  * 节点：古金色圆点；选中节点周围出现墨迹阴影

  * 滚动：横向滚动为主；缩放为可选增强

* 节点概览卡片（点击图表节点弹出）

  * 动画效果：从无到从小到大的缩放弹出（Scale 动效配合 FadeIn）

  * 运势分数：大号醒目字体展示

  * 日记摘要：截取正文前 50 字左右

  * 按钮：“查看详情”，点击跳转到【日记详情页】

***

## Page 3：日记详情（Diary Detail）

### Layout

* 单页阅读沉浸模式，顶部提供修改入口。

### Page Structure

1. 顶部操作栏（返回、修改按钮）
2. 详情内容区（时间、星象、日记正文）

### Sections & Components

* 顶部操作栏

  * 左侧返回按钮。

  * 右侧“修改”按钮，点击后弹出二次确认对话框：“是否确定修改此篇记录？”

* 详情内容区

  * 记录时间：格式 `yyyy-MM-dd HH:mm`

  * 星象快照：显示当前星象状态（如“月亮双子座 | 主要相位：金星拱木星”）

  * 正文区：完整多行文本，保持羊皮纸纹理背景和深色墨迹字体。

***

## Page 4：星盘（Chart）

### Layout

* 全平台：以星盘交互为中心；文案说明尽量少。

### Page Structure

1. 本命盘视图
2. 关键点位摘要（太阳/月亮/上升）

### Sections & Components

* 星盘主体

  * 羊皮纸背景 + 细微纸张纹理；线条为深褐色墨迹

  * 点击/轻触行星点位，仅弹出一句解释（避免面板堆叠）

* 关键点位摘要

  * 以 3 行微文本展示：太阳｜月亮｜上升（与符号）
